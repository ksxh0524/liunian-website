# 流年官网

这是一个极简风格的官网，展示流年（Liunian）产品的核心功能和下载链接。

## 文件结构

```
liunian-website/
├── index.html      # 主页面
├── style.css       # 样式文件
├── script.js       # 交互脚本
├── README.md       # 说明文档
└── assets/         # 资源文件夹（预留）
```

## 功能特点

- ✅ 极简主义设计
- ✅ 响应式布局（支持手机和电脑）
- ✅ 平滑滚动动画
- ✅ 卡片渐入效果
- ✅ 黑白灰主色调 + 蓝色点缀

## 页面结构

1. **Hero Section** - 大标题、Slogan、下载按钮
2. **Features** - 5个核心功能介绍
3. **Why Liunian** - 对比优势展示
4. **Download** - 下载链接和版本信息
5. **About** - 产品故事和联系方式

## 预览

直接在浏览器中打开 `index.html` 即可预览。
